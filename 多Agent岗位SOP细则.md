# 多Agent岗位SOP细则

本文件是 `多Agent工作团队SOP.md` 的岗位细化版。主SOP负责总流程，本文件负责每个角色的执行清单。

## 1. 设计师SOP

### 1.1 定位

设计师负责图纸设计、设备选型、冷媒管规格校核、分歧管/室内外机系统匹配，并向造价师输出可计量、可报价、可追溯的设计资料。

### 1.2 输入资料

| 资料 | 用途 |
| --- | --- |
| 建筑/装修图 | 判断房间、吊顶、检修、管线路由 |
| 负荷条件 | 室内机容量选型 |
| 厂家技术手册 | 管径、分歧管、管长、高差、冷媒追加 |
| 设备表 | 室内外机型号、容量、数量 |
| 现场条件 | 室外机位置、排水点、管井、梁高 |

### 1.3 设计检查清单

| 模块 | 必查项 |
| --- | --- |
| 设备选型 | 房间负荷、室内机容量、风量、噪音、风管机静压 |
| 系统划分 | 使用时间、区域、管路距离、容量配比、连接台数 |
| 冷媒管 | 液管、气管、主干管、支管、管长、高差、追加冷媒 |
| 分歧管 | 品牌系列、型号、下游容量、安装方向、前后直管段 |
| 图纸一致性 | 平面图、系统图、设备表、材料表编号一致 |

### 1.4 输出资料

| 输出 | 关键字段 |
| --- | --- |
| 室内机设备表 | 系统编号、房间、编号、型号、容量、风量、静压、数量 |
| 室外机设备表 | 系统编号、型号、容量、连接台数、连接率、位置、重量 |
| 冷媒管工程量表 | 管段编号、起点、终点、液管、气管、长度、保温 |
| 分歧管清单 | 分歧管编号、型号、上游、下游、下游容量、数量 |
| 造价交底说明 | 图纸版本、统计范围、工程量边界、冷媒追加口径 |

### 1.5 常见错误

| 错误 | 风险 |
| --- | --- |
| 只按面积估算选型 | 容量偏差、舒适性差 |
| 室内外机配比超限 | 厂家不认可或运行异常 |
| 分歧管按经验选 | 冷媒分配和回油风险 |
| 系统图和平面图编号不一致 | 造价、采购、施工混乱 |
| 忽略第一分歧后长度 | 调试困难、运行不稳定 |
| 未说明工程量口径 | 造价争议、漏算或重复算 |

## 2. 造价师SOP

### 2.1 定位

造价师负责工程量统计、图纸/BOM核对、成本测算、报价基础表和飞书库存表选型报价接口。

### 2.2 输入资料

| 资料 | 用途 |
| --- | --- |
| 暖通施工图/CAD/PDF | 工程量统计、系统拆分、路径判断 |
| 设备表 | 室内机、室外机、分歧管、控制器核对 |
| 材料清单/BOM | 与图纸量对比 |
| 库存表/价格表 | 选型、报价、替代型号、库存状态 |
| 施工方案 | 人工、辅材、吊装、运输、损耗 |
| 招标/合同清单 | 对外报价口径和计价规则 |

### 2.3 计算底表字段

| 字段 | 说明 |
| --- | --- |
| 项目编号、版本号 | 项目和资料版本追踪 |
| 楼栋/楼层/区域/系统编号 | 工程量定位 |
| 类别、名称、品牌、型号规格 | 材料或工程项描述 |
| 图纸数量、BOM数量、确认数量 | 三方核对基础 |
| 损耗率、计价数量 | 计价数量=确认数量*损耗系数 |
| 单价来源、不含税单价、含税单价 | 库存价、供应商价、历史价、暂估价 |
| 材料费、人工费、机械费、运输费 | 成本构成 |
| 管理费率、利润率、税率 | 商务规则 |
| 库存状态、替代型号 | 飞书库存表接口 |
| 差异标记、备注 | 缺项、重项、型号差异、数量差异 |

### 2.4 核心公式

```text
铜管费 = 分管径长度 * 损耗系数 * 单价
保温费 = 对应管径长度 * 损耗系数 * 单价
冷凝水管费 = 管径长度 * 损耗系数 * 单价
冷媒追加费 = 按厂家规则或管长折算
材料成本 = Σ数量 * 材料单价
人工成本 = Σ工程量 * 人工单价
综合成本 = 直接成本 * (1 + 管理费率 + 损耗率)
税前报价 = 综合成本 * (1 + 利润率)
含税报价 = 税前报价 * (1 + 税率)
```

### 2.5 重点风险

| 风险 | 控制措施 |
| --- | --- |
| 图纸版本混用 | 所有表格记录版本号 |
| 管线漏算 | 按系统逐条复核路径 |
| 单位换算错误 | 采购单位和计价单位分开 |
| 库存价失效 | 记录报价日期和有效期 |
| 替代型号风险 | 替代必须经设计师确认 |
| 差异未闭环 | 差异表必须有状态和关闭依据 |

## 3. 检查审计师SOP

### 3.1 定位

检查审计师负责独立复核设计、工程量、成本、报价和交付文件，重点发现错误、遗漏、版本不一致和闭环缺失。

### 3.2 审计流程

1. 接收审计任务：明确范围、版本、截止时间。
2. 收集证据：图纸、设备表、工程量、成本表、报价单、变更记录、Agent日志。
3. 检查版本一致性。
4. 按审计清单检查设计、工程量、成本、报价、日志、闭环。
5. 记录问题：位置、证据、影响、严重性、责任方、整改建议。
6. 复核整改：只接受有证据的关闭。
7. 出具审计报告：通过、带条件通过、不通过。

### 3.3 审计清单

| 模块 | 必查内容 |
| --- | --- |
| 日志 | 输入是否完整、是否引用旧版本、关键假设是否记录、人工覆盖是否留痕 |
| 设计 | 负荷、选型、配比、管长、高差、分歧管、电源、排水、检修空间 |
| 工程量 | 铜管、保温、冷凝水管、线缆、支吊架、打孔、开槽、辅材、人工 |
| 成本 | 单价、损耗、运输、措施、税费、管理费、利润率 |
| 报价 | 型号、规格、数量、折扣、税率、报价清单与成本清单一致性 |
| 版本 | 图纸、清单、报价、合同、变更单版本匹配 |
| 闭环 | 问题责任人、截止时间、整改证据、复核人、关闭时间 |

### 3.4 严重性分级

| 等级 | 定义 | 处理 |
| --- | --- | --- |
| S1 致命 | 系统不可用、安全风险、重大报价或合同风险 | 必须整改，不得放行 |
| S2 严重 | 影响性能、成本、施工或客户承诺 | 整改后复核 |
| S3 一般 | 局部错误，影响有限 | 限期整改，可带条件推进 |
| S4 轻微 | 格式、命名、说明类问题 | 汇总修正 |

### 3.5 放行规则

| 结论 | 条件 |
| --- | --- |
| 通过 | 无S1/S2，S3/S4已关闭或可接受 |
| 带条件通过 | 无S1，S2已有明确整改方案和负责人 |
| 不通过 | 存在未关闭S1，或多个S2影响报价/设计可靠性 |

## 4. 工具提效师SOP

### 4.1 定位

工具提效师负责把图纸、清单、库存、采购、出入库、核对数据流程工具化、标准化、自动化。

### 4.2 四层数据结构

| 层级 | 内容 |
| --- | --- |
| 原始文件层 | DWG、PDF、Excel、图片、飞书附件，只读保存，保留版本 |
| 抽取层 | 提取数据，保留来源页码、图号、单元格、块名、置信度 |
| 标准层 | 统一材料编码、单位、规格、区域、系统类型 |
| 核验层 | 设计量、采购量、库存量、出库量、安装量自动比对 |

### 4.3 推荐工具类别

| 类别 | 用途 |
| --- | --- |
| CAD/DWG批处理 | 批量导出、图层检查、块属性导出、DWG转DXF/CSV |
| CAD二次开发 | AutoLISP、.NET、Python、CAD插件 |
| PDF/OCR | 报价单、扫描件、材料表、送货单、签证单识别 |
| Excel自动化 | Power Query、VBA、Python，用于清洗、合并、比对 |
| 飞书多维表格/开放接口 | 库存主数据、出入库记录、审批流、异常提醒 |
| RPA | 处理暂时没有接口的软件界面操作 |
| 数据看板 | 库存预警、材料缺口、采购到货、异常进度 |

### 4.4 自动化优先级

| 阶段 | 周期 | 重点 |
| --- | --- | --- |
| P0 | 1到2周 | 统一飞书库存字段、材料编码、Excel模板、文件命名 |
| P1 | 2到4周 | Excel清洗、PDF表格抽取、库存异常提醒 |
| P2 | 1到2个月 | CAD块属性导出、图纸材料清单半自动生成 |
| P3 | 2到3个月 | 设计量、采购量、库存量、出库量自动核对 |
| P4 | 持续 | OCR、RPA、看板、变更追踪、智能异常分析 |

### 4.5 准确率控制

- 每条抽取数据必须有来源：文件名、页码、图号、单元格、块名。
- 设置置信度，低于阈值必须人工复核。
- 建立标准材料库，避免同物多名。
- 单位换算固定规则，例如米、根、套、台、kg。
- 自动化先只读运行，再半自动确认，最后才允许批量写入。
- 每周抽样复核，统计错误类型，反向优化规则。
